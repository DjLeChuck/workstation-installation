# workstation-installation

Script bash d'installation d'une nouvelle machine sous Linux, type Ubuntu.

## HOWTO
1. Récupérer le script et l'exécuter via `sudo`

    * Avec `curl` : `curl -s https://raw.githubusercontent.com/DjLeChuck/workstation-installation/main/workstation-installation.sh | sudo bash`
    * Avec `wget` : `wget https://raw.githubusercontent.com/DjLeChuck/workstation-installation/main/workstation-installation.sh -q -O - | sudo bash`

1. Cocher les logiciels à installer avec la touche `Espace` puis valide via la touche `Entrée`

    ![Liste des logiciels](./screen-softwares.jpg)

1. Laisser faire et aller se chercher un café ☕
